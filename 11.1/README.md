# Book Inventory

This is a WPF book inventory application created from scratch for the code-first assignment.

## Requirements Covered

- Book inventory table
- ISBN as the primary key
- Book name
- Author name
- Book description
- Additional useful inventory fields: publisher, price, and quantity in stock
- Entity Framework Core code-first approach
- SQLite database created automatically by the application
- Add, update, delete, refresh, and display books

## Run in Visual Studio

Open `Book Inventory.slnx` or `Book Inventory.csproj` in Visual Studio, restore NuGet packages if prompted, and run the project.
