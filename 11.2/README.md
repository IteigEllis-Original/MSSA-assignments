# Assignment 11.2

This console application solves both assignment problems.

## Problem 1

Given an array of stock prices, return the maximum profit from one buy and one future sell. If no profit can be made, return `0`.

The solution keeps track of the lowest price seen so far and the best profit available at each day.

## Problem 2

Given the head of a singly linked list, reverse the list and return the reversed list.

The solution walks through the list once and reverses each node's `Next` pointer.

## Run

Open `Assignment 11.2.csproj` in Visual Studio and run the project.
